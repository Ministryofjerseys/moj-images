# Player Version Jerseys

This folder contains Player Version jersey images for the Ministry of Jerseys website.

## Naming Convention

Format: `{Club/Country}-PlayerVersion-{KitType}-{BatchNumber}{Variant}.jpg`

### Kit Types
- **Home** - Player version home kit
- **Away** - Player version away kit
- **Third** - Player version third kit
- **Special** - Special/Limited player version edition
